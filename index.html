Learning HTML_Section 0

The web
The web is a bunch of computers talking to one another.
Additionally, the web is a collection of HTML (hyper texts markup language).
Major components include; user, computer, internet , http server and browser.

HTML definition
Hypertext-is what the user sees.
Markup- what the layout looks like i.e. tags.
Language- what the computers reads, i.e- images, video, etc.

Markups
Markups are coded in HTML as open and close statements and include < > brakckets.
The totality of these individual statements is called an element.
An element can consist of attributes, images and videos.

Additional Info
White Spac- white space is how much space is seen between each statement and element.
In HTML hitting the space button will not show variation in distance between text or attributes.
Because computers are "stupid" in that they need specific commands, specific code is need to direct computer on how to display statements and elements.
Commands direct computers to follow an inline format, such as break, bold, emphasis, and attributes commands.
Inline commands are commands that have no set boundaries.
Conversly, block commands have specific height, and width parameters.

NOTE
While many of the coding statements in HTML must include closing statements, there are exception i.e the br or break statement.
